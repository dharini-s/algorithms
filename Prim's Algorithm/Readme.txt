1. "make clean" is used to clear the object and other executable files.
2. "make graph_mst" is used to create the executable file.
3. graph_mst takes one argument, which is the file name, eg.
	$./graph_mst dense_100000.txt
4. The output on the console is sum of the edges of the minimum spanning tree and the time taken to execute the program.
4. A file called TestOutput.txt is created, which contains the source vertices, destination vertices and the edges of the minimum spanning tree.
